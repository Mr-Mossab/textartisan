# TextArtisan 🎨

A professional Python library for creating stunning ASCII art and text banners with fonts, colors, gradients, and animations.

## ✨ Features

- 🎯 **Multiple ASCII fonts** (standard, block, slant, digital, fancy, retro)
- 🌈 **Rich color support** (named colors, hex codes, gradients)
- ✨ **Eye-catching animations** (rainbow, blink, typing, scroll, pulse)
- 📦 **Modular design** with clean, intuitive API
- 🖥️ **Powerful command-line interface**
- 🎨 **Decorative borders** and text effects
- 📏 **Automatic centering** and text wrapping
- 💾 **Save to file** functionality
- 🔧 **Extensible** with custom fonts and effects
